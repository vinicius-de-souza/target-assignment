# Target Assignment

## Como executar

1. Clonar o repositório
2. Executar o comando:

```
node main.js
```

## Execução direta

Também é possível executar cada exercício individualmente:

```
node exercise1.js
node exercise2.js
node exercise3.js
node exercise4.js
node exercise5.js
```

## Requisitos

- Node.js
